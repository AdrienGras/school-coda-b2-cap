# Atelier — Analyse du projet

Module **CAP — Concevoir une API REST**. Coda Dijon, B2.

L'Office National des Trajectoires Balistiques vous confie la conception de l'API de
**CATAPULTE Connect**. Avant d'écrire la moindre ligne de contrat, il faut savoir ce que
cette API devra permettre : c'est le travail de cet atelier.

## Ce que vous produisez

[`inventaire.md`](inventaire.md) : la liste des points d'entrée que la plateforme devra
exposer, et des informations qu'ils échangent. En français, sans syntaxe.

Vous le rouvrirez demain — c'est de là que partira votre contrat.

## Vos sources

La **lettre de mission** de l'Office et les **maquettes** du produit, toutes deux sur le site
du cours, avec la consigne détaillée de l'atelier.
