# Atelier — Les fondamentaux HTTP avec curl

Module **CAP — Concevoir une API REST**. Coda Dijon, B2.

Avant de concevoir une API, vous apprenez à en **interroger** une à la main. L'atelier se
joue dans `curl-quests`, un jeu de terminal : chaque quête lance un vrai serveur local,
vous donne une mission, et vérifie votre travail.

## Ce que vous produisez

Le jeu corrige vos commandes ; il ne retient rien pour vous. Votre livrable est
[`aide-memoire.md`](aide-memoire.md) : **votre** fiche de cas, écrite au fil des quêtes.

Une quête résolue, trois lignes à écrire — la situation, la commande qui a marché, ce qui
vous a coincé. Vous la rouvrirez au troisième jour, quand vous taperez sur votre propre
contrat.

## Installer le jeu

Les instructions d'installation, le périmètre des quêtes et le détail de ce qu'il faut
consigner sont dans la fiche d'atelier du site.

Le jeu lui-même : <https://github.com/lite-quests/curl-quests>
